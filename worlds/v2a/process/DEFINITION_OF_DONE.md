
# Definition of Done (DoD)

A chapter can move to `Finished` only if:
- Banned terms check passes (canon/BANNED_TERMS.txt).
- No instructional/lecture paragraphs; concepts are only shown via action/dialogue.
- Nexus does not lecture; only questions/tasks/metaphors.
- Required structure is present: glitch → nudge → test → subtle shift.
- Chapter ends with:
  ---
  SERIES MEMORY UPDATE (for author use)
  - Purpose/Theme: [One sentence summary]
  - factual bullets only
- Continuity check completed; canon updated if new facts introduced.
- PO acceptance recorded (notes).
